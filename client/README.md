# Running the application in Development

1. Using NPM

    ### Install dependencies
    ```bash
    npm i
    ```

    ### Run the application

    ```bash
    npm run start
    ```

2. Using YARN

    ### Install dependencies
    ```bash
    yarn i
    ```

    ### Run the application

    ```bash
    yarn run start
    ```
